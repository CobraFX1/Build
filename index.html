Generators Area
Footprint Dimensions
20 Blocks x 13 Blocks Base
Zoning & Spacing
Positioned along the inner Southeast economy corridor between the Shop Town and the Auction House, providing a central hub for all generator management.
Design Theme
A towering display wall showcase with three vertical generator columns — one for Mining, one for Farming, and one for Slaying. Each column features illuminated item frames showing generator tiers. An example working generator sits on a pedestal at the entrance.

  • ⚡ Example Generator — A working demonstration generator on a raised pedestal so players can see exactly how generators function before purchasing.
